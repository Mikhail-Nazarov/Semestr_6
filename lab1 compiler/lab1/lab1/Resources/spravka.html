О программе.

Данное приложение представляет собой текстовый редактор с реализованными в нем всеми необходимыми функциями для работы с текстом.

Раздел Файл:

  1. Создать - создает новый текстовый файл;
  2. Открыть - открывает текстовый файл;
  3. Сохранить - сохраняет редактируемый текстовый файл;
  4. Сохранить как - сохраняет редактируемый текстовый файл с возможностью выбора места для сохранения;
  5. Выход - закрытие приложения.

Раздел Правка:

  1. Отменить - отменяет последнее изменение;
  2. Повторить - возвращает отмененное действие;
  3. Вырезать - вырезает выделенный текст;
  4. Копировать - копирует выделенный текст;
  5. Вставить - вставляет скопированный текст;
  6. Удалить - удаляет выделенный текст;
  7. Выделить все - закрытие приложения.

О регулярных выражениях.

Регулярные выражения в данном приложении позволяют найти все фамилии с инициалами в тексте, а также строку, в которой встретилась фамилия, и номер символа, с которого начинается фамилия. Фамилия может быть двойной, инициалы могут быть расположены до и после фамилии. Фамилия и инициалы должны иметь вид: Римский-Корсаков Н.А. или А.С. Пушкин.

О конечном автомате.

Заглавные русские буквы (А-Я), Строчные русские буквы (а-я), Пробел, Точка, Дефис, Запятая.
